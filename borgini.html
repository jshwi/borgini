borgini¶

borgini¶

A wrapper to quickly get you started backing up with borg

An easy to use ini style and profile based format

See README for examples or run borgini --help for usage

class borgini.Data(appdir, profile)[source]¶

Bases: object

Resolve the directory the configurations belong to. Get the list of files to include and exclude. Allow the files to contain comments and avoid parsing them.

Parameters

profile – The profile that borgini is being run under.

get_exclude()[source]¶

Return the values obtained from the exclude file

Unlike the include file this doesn’t necessarily need to be populated

Every item will automatically be prefixed with --exclude for the borgbackup commandline

Returns

A lit of paths to exclude - this will override items in include
